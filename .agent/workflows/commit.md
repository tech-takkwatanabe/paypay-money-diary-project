---
description: 変更のあるファイルを1件ずつ確認し、gitmojiと日本語メッセージでコミットを行います。
---

下記のステップに従って、ユーザーと対話しながらコミットを進めてください。

1. **変更の確認**: `git status --porcelain` を実行し、変更のあるファイル（修正、新規、削除、未追跡）を特定します。
2. **コミット情報の入力**: 変更のある各ファイルについて、gitmoji（例: :sparkles:, :bug:）と簡潔な日本語メッセージ、Issue番号（あれば）を入力または提案します。
3. **コミットの実行**: メッセージが決まり次第、**確認ステップを挟まずに**即座に `git add <file>` と `git commit -m ":emoji: #issue番号 内容"` を実行します。
4. **完了**: すべての対象ファイルの処理が終わるまで繰り返します。
